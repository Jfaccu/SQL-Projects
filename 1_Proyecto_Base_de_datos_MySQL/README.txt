			PUNTOS A TENER EN CUENTA A LA HORA DE 
			EJECUTAR LOS SCRIPT DE LA BASE DE DATOS.

	En En documento encontraremos la carpeta que contiene todos
	los script de la base de datos y los objetos que la componen.
	En primer instancia los script estaran enumerados respectivamente
	en la forma que tiene que ser ejecutados:

0- Create Restauran_BD = Este script a la hora de ejecutarlo se creara la estructura
de la base de datos.

1- Script Insert Datos = Al correr este codigo se llenaran las tablas con dicha
información para cada tabla.

2- Create View = Al crear las vistas, podremos ver consultas que necesitaremos
concurrentementede de la base de datos.

3- Funciones = Es solo un script en donde apreciaremos las funciones generadas.

4- Store Procedure = Se añadiran a la base de datos Procesos almacenados para el Restó.

5- Triggers y Tablas Log = Este Script contiene los dos disparadores para ambas tablas,
tanto Clientes y Ordenes.

6- Sentencias DCL = Codigo para otorgarle acceso a usuarios a la base de datos.

7- Sentencias TCL = Script del lenguaje Transaccional Control Language para las tablas 
Ordenes y Clientes.
